# My VIM Source File

Follow those steps...
-------------------------------

1. You gotta download Vundle (Plugin manager)
  $ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

  So simple.

2. Open vim and then :PluginInstall
  
  Will install all plugins setted on .vimrc

3. Have Fun!
